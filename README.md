# LeftMenueDemo
左滑菜单、抽屉效果，并封装了Tabbar和Navigation导航控制器

1、实现左滑滑动、返回滑动、点击返回的手势功能<br>
2、封装Navigation导航栏控制器，自动管理左滑手势的打开与关闭(只在首页的时候打开手
势)<br>
3、如果不需要TabbarController，只需要把TabbarController换为自己的根控制器即可<br>
